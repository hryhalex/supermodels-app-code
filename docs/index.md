# Цветочная оранжерея
## Командный проект (вариант 8)

### Команда: Supermodels
**Состав:**
- Кудинова Елизавета — разработчик
- Григорьева Александра — тимлид

---

### Описание проекта
Консольное приложение для управления цветочной оранжереей. Система позволяет:
- Управлять каталогом растений (добавление, редактирование, удаление, просмотр)
- Создавать композиции из растений
- Обрабатывать заказы клиентов
- Хранить фотографии растений
- Разграничивать права доступа (admin, manager, customer)

---

### Функциональные возможности
- Регистрация новых видов растений в системе
- Управление композициями (создание, состав, стоимость)
- Обработка заказов (создание, завершение, поиск)
- Статистика и отчёты (выручка, популярные композиции)
- Хранение фотографий растений (пути к файлам)

---

### Технологический стек
- **Язык:** C / C++ (C11, C++17)
- **База данных:** SQLite 3
- **Тестирование:** Google Test (15 unit-тестов)
- **Сборка:** Makefile
- **CI/CD:** GitHub Actions (автоматическая сборка и тестирование)
