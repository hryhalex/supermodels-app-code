# Командный проект: Цветочная оранжерея (Вариант 8)
**Команда:** Supermodels  
**Состав:** Кудинова Елизавета, Григорьева Александра  

## Описание проекта
Приложение предназначено для автоматизации учета растений в оранжерее, контроля условий их содержания и ведения базы данных складских запасов.

## Функциональные возможности
* Регистрация новых видов растений в системе.
* Мониторинг температуры и влажности для каждой зоны.
* Учет периодичности полива.
* Хранение и просмотр фотографий растений (BLOB).

## Технологический стек
* **Язык:** C (стандарт C11).
* **База данных:** SQLite 3.
* **Автоматизация:** Makefile + GitHub Actions.
