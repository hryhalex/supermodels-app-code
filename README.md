# Цветочная оранжерея

## Project Name
**Flower Greenhouse Management System**

## Description
Консольное приложение для управления цветочной оранжереей. Система позволяет:
- Управлять каталогом растений (добавление, редактирование, удаление, просмотр)
- Создавать композиции из растений
- Обрабатывать заказы клиентов
- Хранить фотографии растений
- Разграничивать права доступа (admin, manager, customer)

## Installation

### Требования
- macOS / Linux / Windows (с WSL)
- GCC / Clang
- SQLite3
- Google Test (для запуска тестов)

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/hryhalex/supermodels-app-code.git
   cd supermodels-app-code
   ```

2. Установите зависимости (macOS):
   ```bash
   brew install sqlite3 openssl
   ```

3. Создайте базу данных:
   ```bash
   sqlite3 data/greenhouse.db < data/schema.sql
   ```

4. Соберите проект:
   ```bash
   make clean
   make
   ```

## Usage

### Запуск
```bash
make run
```

### Тестирование
```bash
make test
```

### Очистка
```bash
make clean
```

### Учётные записи
| Логин | Пароль | Роль |
|-------|--------|------|
| admin | admin | Полный доступ |
| manager | 123456 | Управление растениями, композициями, заказами |
| customer | 123456 | Только просмотр |

### Основные функции
- **Управление растениями** — просмотр, добавление, редактирование, удаление
- **Управление композициями** — создание композиций, добавление/удаление растений
- **Управление заказами** — создание, редактирование, завершение, поиск по статусу/клиенту/дате
- **Статистика** — выручка, популярные композиции, заказы по срочности
- **Изображения** — пути к фотографиям растений хранятся в БД
